SVG

Scaled Vector Graphics (SVG) files allow the definition of scalable images.

Scalability includes a number of different transformations. For the current
implementation, we only consider size and rotation. Matrix and skew transforms
will be added if there is a need.

Image size parameters are applied to the <svg> element. As the rotation
transformation on the <svg> element is only defined in SVG 2,
transformations will be applied to a <g> element that contains all of the
graphics in the <svg> element.

Single sub-element SVG

For instance, if the following SVG file were to be displayed as 108px x 112px
and rotated 20 degrees about its center 

<svg version="1.0" xmlns="http://www.w3.org/2000/svg"
        viewBox="0 0 54.000000 56.000000"
        preserveAspectRatio="xMidYMid meet">
    <g fill="#000000" stroke="none">
        <path d="M133 503 ... -6z"/>
    </g>
</svg>

it would be transformed to:

<svg version="1.0" xmlns="http://www.w3.org/2000/svg"
        width="108pt" height="112pt" viewBox="0 0 54.000000 56.000000"
        preserveAspectRatio="xMidYMid meet">
    <g transform="rotate(20,27,28)"
        fill="#000000" stroke="none">
        <path d=" ... "/>
    </g>
</svg>

Multiple sub-element SVG

If there are multiple elements directly below the <svg> element, they are
enclosed in a new <g> element. 

<svg version="1.0" xmlns="http://www.w3.org/2000/svg"
        viewBox="0 0 54.000000 56.000000"
        preserveAspectRatio="xMidYMid meet">
    <g fill="#000000" stroke="none">
        <path d=" ... "/>
    </g>
    <g>
        <rect x=.../>
    </g>
</svg>

it would be transformed to:

<svg version="1.0" xmlns="http://www.w3.org/2000/svg"
        width="108pt" height="112pt" viewBox="0 0 54.000000 56.000000"
        preserveAspectRatio="xMidYMid meet">
    <g transform="rotate(20,27,28)">
        <g fill="#000000" stroke="none">
            <path d=" ... "/>
        </g>
        <g>
            <rect x=.../>
        </g>
    </g>
</svg>

Added attributes

The added attributes are:

<svg> width and height
Set tospecified values.
If width and height are not specified, they are derived from the viewbox values.
<g> transform
The rotation transformation.

Dynamic transformations

If the size and rotation are to vary, the <svg> content is altered programmatically.
